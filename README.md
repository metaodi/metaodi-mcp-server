# metaodi-mcp-server
MCP server for my tools like OpenERZ or tecdottir.

## Usage

Python SDK: https://github.com/modelcontextprotocol/python-sdk

```
uv run mcp
```

Run OpenERZ:

```
uv run mcp run openerz.py
```


MCP Inspector:

```
npx -y @modelcontextprotocol/inspector uv run mcp run openerz.py
```
